# 🕒 Digital Clock

A simple and responsive Digital Clock built using **React.js** and **Tailwind CSS**. The clock displays the current time in **HH:MM** format and updates automatically every second.

---

## 🚀 Features

- 🕒 Displays the current time in HH:MM format
- 🔄 Updates automatically every second
- 🎨 Modern and responsive UI
- ⚛️ Built with React functional components
- 💨 Styled using Tailwind CSS

---

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- JavaScript (ES6)
- Vite

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/digital-clock.git
```

2. Navigate to the project folder

```bash
cd digital-clock
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open the URL displayed in the terminal (usually `http://localhost:5173`).
